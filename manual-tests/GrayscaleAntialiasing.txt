Grayscale antialiasing test
===========================

#. Launch gnome-tweak-tool and select "Fonts" in the left pane
#. Select value "Grayscale" for "Antialiasing"
#. Log out and back in

Outcome
 Text in unity panel, launcher and lenses should be antialiased using grayscale method.
 If text is not antialiased, the test is failed.
